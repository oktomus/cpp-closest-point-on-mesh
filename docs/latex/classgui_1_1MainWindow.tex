\doxysubsection{Detailed Description}
Standalone application. 

G\+L\+FW is used to handle the window and the events. Dear imgui is used to draw some UI elements.

This standalone application allow to load 3D models and run the \char`\"{}closest point on mesh\char`\"{} algorithm on them.
